# decisions/

Local decisions at Archon scope. See canonical conventions at `/Volumes/G-DRIVE/Shared/SecurityByDesign-TheoryOfEverything/agentic/decisions/README.md`.

**Not** programme-level ADRs — those live in `DevelopmentGovernance/docs/decisions/`. Ontology-formalization decisions (e.g. ADR 0005, 0010) are programme-level and go to the governance repo.

Examples of decisions that **do** live here:
- CO promotion or deferral rationale (before it becomes a programme-level ADR)
- Slice boundary interpretation on a specific source item
- OWL export pipeline changes that don't rise to ADR level
- Identifier-stability exceptions (none expected — stability is contractual)
