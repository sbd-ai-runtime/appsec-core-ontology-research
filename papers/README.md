# Papers

This directory is organized paper-by-paper. Each paper has its own self-contained publication surface:

- `source/` for the curated manuscript source
- `pdf/` for the public reading PDF
- `arxiv/` for the arXiv submission bundle (`main.tex` and `.tar.gz`)
- `artifacts/` for the curated supporting materials referenced by that paper

The current `V1` papers are:

- `01-appsec-core-normalized-ontology/`
- `02-coverage-preserving-knowledge-compilation/`
- `03-ontology-grounded-retrieval/`
