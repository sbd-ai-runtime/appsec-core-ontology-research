# Papers

This directory is organized paper-by-paper. Each paper has its own self-contained publication surface:

- `source/` for the curated manuscript source
- `pdf/` for the public reading PDF
- `arxiv/` for the arXiv submission bundle (`main.tex` and `.tar.gz`)
- `artifacts/` for the curated supporting materials referenced by that paper

The current public `v1` tree contains:

- `01-appsec-core-normalized-ontology/`
- `02-coverage-preserving-knowledge-compilation/`
- `03-ontology-grounded-retrieval/`
- `04-empirical-evaluation/` — frozen OSF-authoritative mirror
- `05-mcp-se-engineering/` — frozen companion apparatus mirror

Interpretation:

- `01` to `03` are the core `V1` paper set
- `04` is included as a frozen companion mirror of the OSF-registered design, not as part of the core three-paper `V1` claim set
- `05` is included as a frozen companion mirror of the apparatus-specification manuscript that supports the `P4` study, not as part of the core three-paper `V1` claim set
